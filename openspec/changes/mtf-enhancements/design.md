## Context

MSX Tile Forge support was added in release 0.3.3.3 (CMD MTF). This sprint enhances the feature based on user feedback and discovered use cases.

## Goals / Non-Goals

**Goals:**
- Improve MTF map loading performance
- Add new MTF features and commands

## Decisions

Optimize map loading before adding new features.

## Risks / Trade-offs
